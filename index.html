🗺️ How to Add a New Daily Report

Step 1 — Generate JSON via Claude

  1. Open claude.ai in a new tab
  2. Upload your NEPSE PDF report
  3. Copy the prompt below and paste it to Claude
  4. Claude returns a JSON object

Step 2 — Upload to GitHub Repo

  1. Go to github.com/SagunPrajapati/share
  2. Click Add file → Create new file
  3. Name it YYYY-MM-DD.json (e.g. 2026-04-17.json)
  4. Paste the JSON → click Commit changes
  5. Click 🔄 Refresh Dates on the frontend
